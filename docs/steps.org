#+title:  How to generate HTML using Org
#+SETUPFILE: ./org-templates/level-0.org 


* Introduction
  This document describes how to generate HTML files and apply CSS and
  JavaScript using Org.
* Requirements
 - Org-mode version 8.3.4
 - Emacs 24.4.1
 - prj.el
* Installing a theme
** level-0.org

In order to use the =level-0= style, just copy the [[file:../src/org-templates/level-0.org][level 0]] file to
your project and link to it this way:
#+BEGIN_EXAMPLE
#+SETUPFILE: ./org-templates/level-0.org
#+END_EXAMPLE

If you prefer, you can add the content of the setup file at the beginning of
your Org file instead:

#+BEGIN_EXAMPLE
#+HTML_HEAD:  <link rel="stylesheet" type="text/css" href="style/css/bootstrap.min.css" />
#+HTML_HEAD_EXTRA:  <link rel="stylesheet" type="text/css" href="style/css/main.css" />
#+END_EXAMPLE

** Local version

Here a local version (using relative paths):
#+BEGIN_EXAMPLE
#+HTML_HEAD:  <link rel="stylesheet" type="text/css" href="style/css/bootstrap.min.css" />
#+HTML_HEAD_EXTRA:  <link rel="stylesheet" type="text/css" href="style/css/main.css" />
#+HTML_HEAD_EXTRA:  <link rel="stylesheet" type="image/png" href="style/img/banner_img.png" />
#+END_EXAMPLE
